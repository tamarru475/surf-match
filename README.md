# surf-match
SurfMatch helps New Zealand surfers find the right wave by matching their skill level, preferences, and local surf conditions to suitable surf spots.

This project is still in the planning phase you can check out the architecture for V1 in this Figma file: https://www.figma.com/make/1zPtZDVsRfjrsyEbO38UZG/SurfMatch-app-architecture?viewport=298%2C384%2C1&t=nQaJXXoqpd7uV4Td-0
